# ROSSERIAL-ROBOT
 * ROS with Arduino Robot DC Motors Control

## Usage
* All the usage is commented with a code

## Technology
 * Arduino IDE
 * ROS Noetic
 * Ubuntu 20.4
 * KKMOON Robot chassis
 * Arduino UNO 3
 * Arduino Niady L293D Motor Shield
 * A pair of 18650 batteries
 * Wires
